# 瑞莱生物 - 生产管理系统

## 项目概述

React Native + Expo Android 移动端应用，工业风深色 UI。

## 技术栈

- Expo SDK 55 / React Native 0.83 / TypeScript 5.9
- 函数组件 + StyleSheet
- Expo Go 调试运行

## 开发规范

- 使用 TypeScript 严格模式
- 函数组件 + Hooks
- 样式使用 StyleSheet.create
- 注释使用中文
- Git 提交格式：`feat:` / `fix:`

## 常用命令

```bash
npm install          # 安装依赖
npx expo start       # 启动开发服务器
npx expo start --clear  # 清除缓存启动
```

## 项目结构

- `App.tsx` — 应用入口
- `assets/` — 静态资源
